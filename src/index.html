• 每一个标签
  • 都加了上自定义属性 xusheng

css文件如下：

        
        div {
            margin: 0 auto;
            width: 1000px;
            font-size: 1.5em;
        }
        div.wrap ul.list li.item {
            transition: all 1s;
        }

        div.wrap ul.list li.item:hover {
            color: red;
            font-size: 1.2em;
        }

        div#article > ul ~ li {
            font-weight: bold;
        }

        div.demo {
            border: 1px solid #ccc;
            box-shadow: 1px 1px 4px #444;
        }

        .title {
            text-indent: 2em;
            color: red;
        }

打包后css文件如下：

        
        div[xusheng] {
            margin: 0 auto;
            width: 1000px;
            font-size: 1.5em;
        }
        div.wrap[xusheng] ul.list[xusheng] li.item[xusheng] {
            transition: all 1s;
        }

        div.wrap[xusheng] ul.list[xusheng] li.item:hover[xusheng] {
            color: red;
            font-size: 1.2em;
        }

        div#article > ul[xusheng] ~ li[xusheng] {
            font-weight: bold;
        }

        div.demo[xusheng] {
            border: 1px solid #ccc;
            box-shadow: 1px 1px 4px #444;
        }

        .title[xusheng] {
            text-indent: 2em;
            color: red;
        }
